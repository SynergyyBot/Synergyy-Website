Features

Learn about what Synergyy has to offer. Scroll down to the commands list to see how to use these features!

Synergyy is an easy to use project management Discord bot built for students. With features such as Meeting Creation and Reminders, To-Do Lists, Polls, and more, Synergyy allows students to easily collaborate with others on Discord!

Create Team Meetings

Easily set up team meetings at certain times and be reminded about them before they start!

Create To-Do Lists

Plan out your group's tasks, and check off tasks as you complete them.

Create Polls

Want to vote on the best way to approach your project? Set up a dynamic poll with ease!
