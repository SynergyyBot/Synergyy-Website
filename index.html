Features

Learn about what Synergyy has to offer. Scroll down to the commands list to see how to use these features!

Synergyy is an easy to use project management Discord bot built for students. With features such as Meeting Creation and Reminders, To-Do Lists, Polls, and more, Synergyy allows students to easily collaborate with others on Discord! The bot was built using Discords Python API (discord.py).

Create Team Meetings

Easily set up team meetings at certain times and be reminded about them before they start!

Create To-Do Lists

Plan out your group's tasks, and check off tasks as you complete them.

Create Polls

Want to vote on the best way to approach your project? Set up a dynamic poll with ease!

Commands List

Check out the cards below to see the full list of commands that Synergyy offers!
In Discord, this list can be called using !help.

!meeting

Creates a new meeting.
• !meeting "Physics Project" in 2 hours
• !meeting "Team Call" on September 1 at 11:00 AM
Use !list to see your meetings and use !delete to delete them.

!todo and !complete

Creates a todo list and check off items once complete.
• !todo "Finish English Powerpoint"
The !complete complete command will open a screen that allows you to check off items.

!poll

Creates a dynamic poll.
• !poll "Title" options (time limit in minutes)
• !poll "Favorite Food?" Taco, Pizza, Sushi 2

!timenow

Lists the current times across the globe.
This includes every major timezone, which makes it easier to coordinate meetings with international students.

!clear

Clears a certain number of messages from the Discord channel.
• !clear 50
If no number is given, the default is 10 messages.

!vote

Vote for Synergyy on top.gg to help us grow!
Note: You can vote once every 12 hours, and votes are worth double on weekends.

!8ball

Ask the magical 8ball a question!
• !8ball Will I pass this test?

!flip

Flip a coin!
It's an easy way to settle decisions between team members!
